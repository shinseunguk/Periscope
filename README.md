# Periscope

A Swift library for iOS, macOS, tvOS, and watchOS.

## Installation

### Swift Package Manager

Add the following to your `Package.swift` file:

```swift
dependencies: [
    .package(url: "https://github.com/shinseunguk/Periscope.git", from: "1.0.0")
]
```

Or add it through Xcode:
1. File â†’ Add Packages...
2. Enter: `https://github.com/shinseunguk/Periscope.git`
3. Click "Add Package"

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'Periscope', '~> 1.0.0'
```

Then run:
```bash
pod install
```

## Requirements

- iOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- watchOS 6.0+
- Swift 5.7+
